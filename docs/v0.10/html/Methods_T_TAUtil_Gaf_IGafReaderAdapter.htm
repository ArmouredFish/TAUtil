IGafReaderAdapter Methods

The IGafReaderAdapter type exposes the following members.

Methods
  NameDescription
BeginEntry
Indicates that the reader has begun reading a new GAF entry.
BeginFrame
Indicates that the reader has begun reading a new frame in the current entry. If we were already reading a frame, this new frame is a subframe of the current frame.
BeginRead
Indicates that reading of the GAF file has started.
EndEntry
Indicates that the reader has finished reading the current entry.
EndFrame
Indicates that the reader has finished reading the current frame.
EndRead
Indicates that the reader has finished reading the file.
SetFrameData
Sets the image data for the current frame. This will be called only if the frame contains no subframes.
