TAUtil.Tdf Namespace
Utilities for working with TDF format files. TDF is a text-based format used for storing property/attribute or configuration data. It provides a hierarchical key-value pair structure for storing information.
Classes
  ClassDescription
TdfNode
Tree structure representing a hierarchy of keys and values as in a TDF format file.
TdfNodeAdapter

Adapter for the TdfParser class.

This adapter constructs a tree of TdfNode objects from the parsed TDF file.

TdfParser
Class for parsing TDF format files.
Interfaces
  InterfaceDescription
ITdfNodeAdapter
Adapter for the TdfParser class. Users of TdfParser should implement this interface to control how TDF records are processed during parsing.
