ISctSource Methods

The ISctSource type exposes the following members.

Methods
  NameDescription
EnumerateAttrs

Enumerates the attribute data. Each cell contains a TileAttr structure indicating the attributes for the cell. The attributes are enumerated row by row, starting from the top left. The length of the data is DataWidth * DataHeight * 4. (There are four attribute cells for each data cell.)

In the case of SCT files, only the height data from each attribute is written.

EnumerateData
Enumerates the tile data. Each cell contains an index indicating which tile in the tile list should be displayed at this location. The data is enumerated row by row, starting from the top left. The length of the data is DataWidth * DataHeight.
EnumerateTiles
Enumerates the tile list. Each tile is an array of binary data.
GetMinimap
Gets the section's minimap. This is a 128x128 indexed color image. The length of this data in bytes is 128*128.
