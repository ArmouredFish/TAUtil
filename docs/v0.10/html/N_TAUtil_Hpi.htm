TAUtil.Hpi Namespace
Utilities for reading and writing HPI format files. HPI is the packed archive format used within Total Annihilation. A HPI archive is similar to a zip file in that it contains a virtual directory structure in a packed format.
Classes

HpiEntry
Represents a file a directory entry inside a HPI archive.
