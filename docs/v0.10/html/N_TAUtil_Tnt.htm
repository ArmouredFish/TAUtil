TAUtil.Tnt Namespace
Utilities for working with TNT format files. TNT is the format used for map terrain data within Total Annihilation. It contains the map graphics, heightmap and feature information.
Classes
  ClassDescription
MinimapInfo
Data structure containing information about the minimap for a map or section.
TntConstants
Contains constants relevant to TNT format files.
TntReader
Class for reading from TNT format files.
TntWriter
Class for writing TNT format files to a stream.

Interfaces
  InterfaceDescription
ITntSource

Adapter for the TntWriter class. This interface provides properties and methods for retrieving data in order to write it to a TNT file.

Users should implement this interface to allow TntWriter to extract the necessary data for writing.
