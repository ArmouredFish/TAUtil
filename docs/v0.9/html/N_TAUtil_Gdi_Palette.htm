TAUtil.Gdi.Palette Namespace
Utilities for working with indexed color palettes.
Classes
  ClassDescription
ArrayPalette
Indexed color palette backed by a fixed size array.
BinaryPaletteReader
Wrapper for reading TA palette files. The files themselves are just 256 colors packed in binary, 4 bytes for each color: r, g, b, a.
PaletteFactory
Provides methods for creating palette instances from files.
Quantization
Provides methods for quantizing images to the TA color palette.
TransparencyMaskedPalette
Wrapper for an IPalette instance that provides support for using one of the index values as a transparency mask.
Interfaces
  InterfaceDescription
IPalette

Indexed color palette interface.

Provides methods for indexed color lookups, reverse lookups and nearest neighbour searches.
