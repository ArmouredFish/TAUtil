Enumerates the files at the root of the HPI archive.

This enumeration is not recursive, so files inside directories will not be included.

Beware that paths inside HPIs can contain special characters such as ">", which will cause methods in Path to throw an exception. To manipulate paths originating from HPI archives, use HpiPath.
